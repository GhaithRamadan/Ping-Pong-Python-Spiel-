# Ping Pong Game 

Ein **kleines und eigenständiges Projekt**, das ich zur praktischen Anwendung meiner Grundlagen in Python entwickelt habe.

## Funktionen
- Zwei Spieler Steuerung über die Tastatur  
- Ballbewegung mit Kollisionsabfragen  
- Punktesystem mit Anzeige des Spielstands  

## Technologien
- Python  
- Turtle  

## Ziel des Projekts
Ziel dieses Projekts war es, grundlegende Konzepte der Programmierung wie Logik, Zustandsänderungen und Ereignissteuerung praktisch umzusetzen.
